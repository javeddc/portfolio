LocalPianos (beta)

Project Description:

Many pianos are chronically under-played. They sit in cafes, libraries, airports, and other public spaces, while many people who can play them can't commit to owning one.

LocalPianos maps these publicly accessible pianos, and allows users to add new pianos they find. A React-based first release is currently in development, which will also allow users to comment, add pictures, and embed audio samples of each piano.

This beta was produced in a group project at General Assembly, with Brett Cremer and Alex May working on the front end.

Built with:
Rails Geocoder jQuery CSS

Disnumber

Project Description:

Never forget a phone number again!

Disnumber is a web app that allows you to turn any phone number into a memorable two-word phrase. You can then give this phrase to others to look up instead of remembering or exchanging long numbers.

Disnumber was built as an early project in the General Assembly Web Development Immersive.
